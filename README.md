# 💸 Guardian Financiero

¡Bienvenido! Este es nuestro asistente personal de finanzas diseñado para mejorar nuestra salud financiera en pareja.

## 🚀 ¿Qué hace esta app?
* **Registro con IA:** Clasifica gastos automáticamente usando Google Gemini.
* **Sincronización:** Conectada a Firebase para ver los gastos de ambos en tiempo real.
* **Control:** Dashboard interactivo para ver cuánto nos queda antes de la siguiente quincena.

## 🛠️ Tecnologías
* React + TypeScript
* Firebase (Auth & Firestore)
* Google AI Studio (Gemini API)
